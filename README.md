Система автоматизирует процесс записи клиентов на приём к ветеринару. Реализован полный пользовательский сценарий:
Регистрация → Авторизация → Создание заявки → Просмотр истории → Оставление отзыва → Панель администратора
Проект построен на трёхуровневой архитектуре: Frontend (HTML/CSS/JS) ↔ Backend (Flask) ↔ Database (SQLite).
Реализованный функционал
-Авторизация и регистрация с валидацией, хешированием паролей (werkzeug.security) и синхронизацией профилей
-Личный кабинет: таблица заявок, фильтр по дате/услуге, кнопка создания отзыва
-Создание заявки: выпадающие списки из БД, проверка времени (9–18), AJAX-отправка через fetch
- Панель администратора (/admin): доступ только для ролей admin/директор, отображение только текущих и будущих заявок
- Слайдер на главной: автопрокрутка, стрелки, адаптивные пропорции (aspect-ratio, object-fit: cover)
- Toast-уведомления и микроанимации (hover, fade, loader)
- Доступность: label/for, alt, контраст ≥ 4.5:1, навигация с клавиатуры, Lighthouse ≥ 90
Технологический стек
Backend: Python 3.10+, Flask, werkzeug, sqlite3
Frontend: HTML5, CSS3 (Grid/Flex, CSS Variables), Vanilla JS (Fetch API)
База данных: SQLite3 (PRAGMA FK, транзакции, индексы)
Инструменты: Git, Lighthouse, W3C Validator, DevTools
